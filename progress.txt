# CIRO Project Progress

## Tech Stack
- **Backend Orchestrator**: Python, FastAPI, SQLite, LLM APIs (Gemini/Claude)
- **Mobile App**: React Native, SQLite Storage, BLE PLX (Bluetooth Mesh)

## Core Concepts
- **Agentic Orchestration**: Modular AI agents handling specific parts of the crisis response lifecycle.
- **Offline-First Resilience**: Mobile app operates independently when internet is down using local cached models.
- **Ad-Hoc Mesh Networking**: Encrypted messages relayed via Bluetooth between mobile devices until a cloud connection is found.
- **Signal Fusion**: Combining multiple data sources (social media, weather, sensors) and detecting contradictions to prevent false alarms.
